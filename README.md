# 🏎️ RM Motosport

Site institucional desenvolvido para empresa de motosport, apresentando serviços, galeria e formulário de contato.

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e responsividade
- **JavaScript** - Interatividade e validações

## ✨ Funcionalidades

- ✅ Design responsivo para mobile e desktop
- ✅ Carrossel de imagens
- ✅ Formulário de contato com validação
- ✅ Menu de navegação interativo
- ✅ Galeria de fotos

## 📦 Como Executar

1. Clone o repositório:
git clone https://github.com/Arthur-Bjr/RMmotosport.git

2. Navegue até a pasta:
cd RMmotosport

3. Abra o arquivo index.html no navegador:
   - Dê duplo clique no arquivo, OU
   - No VS Code: botão direito → Open with Live Server
